GIt + hexo 搭建博客

一、 介绍

  • Hexo是一款基于Node.js的静态博客框架

搭建步骤

  1. 安装git 和node.js环境

  2. 全局安装hexo

  3. 初始化hexo

    在放博文的路径

  4. hexo s 启动服务

  1. 将hexo远程关联GitHub

    修改站点配置文件_config.yml

    这一步的作用是给hexo d命令做相应的配置

  2. 保存配置文件后执行以下命令

  3. 发布文章

    在目录下即可查看到对应的md文件

  4. 更换主题

  • 在博客的文件夹内的themes目录下git clone主题

  • 修改站点配置文件

    Set theme in the hexo work folder’s root config file _config.yml:

theme: butterfly

  • 更改之后重新生成并部署

常用hexo命令：
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署
